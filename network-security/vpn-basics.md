# VPN Basics

## Overview
A Virtual Private Network (VPN) creates a secure, encrypted tunnel between a user and a remote server. It protects data from eavesdropping and hides the user's IP address.

## Why VPNs Are Used
- Secure remote access
- Protect data on public Wi-Fi
- Hide IP address and location
- Bypass geo-restrictions
- Prevent ISP tracking

## Types of VPNs
### 1. **Remote Access VPN**
Used by individuals to securely connect to a private network.

### 2. **Site-to-Site VPN**
Connects two networks (e.g., branch office to headquarters).

### 3. **Client-to-Site VPN**
Employees connect to company resources from home.

## Common VPN Protocols
- **OpenVPN** – secure, open-source, widely used  
- **IPSec** – used for site-to-site VPNs  
- **L2TP** – often combined with IPSec  
- **WireGuard** – modern, fast, lightweight  
- **IKEv2** – stable on mobile devices  

## Key Concepts
- Encryption  
- Tunneling  
- Authentication  
- Key exchange  
- Split tunneling  

## What I Learned
- VPNs protect data by encrypting traffic  
- Different protocols offer different levels of speed and security  
- VPNs are essential for secure remote work
