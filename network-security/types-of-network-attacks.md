# Types of Network Attacks

## Overview
Network attacks are attempts to disrupt, steal, or gain unauthorized access to data or systems. Understanding these attacks is essential for defending networks.

## 1. Denial of Service (DoS)
- Overwhelms a system with traffic  
- Makes services unavailable  
- Example: SYN flood

## 2. Distributed Denial of Service (DDoS)
- Multiple systems attack a target  
- Harder to block  
- Often uses botnets

## 3. Man-in-the-Middle (MITM)
- Attacker intercepts communication  
- Can steal credentials or modify data  
- Example: ARP spoofing

## 4. Phishing & Social Engineering
- Tricks users into revealing sensitive information  
- Often uses fake emails or websites

## 5. Spoofing Attacks
- Attacker pretends to be a trusted device  
- Types: IP spoofing, MAC spoofing, DNS spoofing

## 6. Port Scanning & Reconnaissance
- Used to discover open ports and services  
- Often the first step before an attack

## 7. Malware Attacks
- Includes viruses, worms, ransomware, trojans  
- Can steal data or damage systems

## 8. SQL Injection
- Injects malicious SQL into a database query  
- Can extract or modify data

## What I Learned
- Network attacks target availability, integrity, and confidentiality  
- Many attacks begin with reconnaissance  
- Defense requires layered security and monitoring
