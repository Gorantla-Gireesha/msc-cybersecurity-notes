# OSI Model

## Overview
The OSI (Open Systems Interconnection) model is a conceptual framework that standardizes the functions of a networking system into seven layers. It helps understand how data travels across a network.

## The 7 Layers of OSI Model
1. **Physical Layer**  
   - Deals with cables, signals, bits  
   - Examples: Ethernet cables, Wi-Fi signals

2. **Data Link Layer**  
   - Responsible for MAC addresses, frames, error detection  
   - Examples: Switches, ARP

3. **Network Layer**  
   - Handles IP addressing and routing  
   - Examples: Routers, IPv4/IPv6

4. **Transport Layer**  
   - Ensures reliable or fast delivery  
   - Protocols: TCP, UDP

5. **Session Layer**  
   - Manages sessions between devices  
   - Examples: API sessions, authentication sessions

6. **Presentation Layer**  
   - Data formatting, encryption, compression  
   - Examples: SSL/TLS, JPEG, MP4

7. **Application Layer**  
   - Closest to the user  
   - Examples: HTTP, DNS, FTP, SMTP
