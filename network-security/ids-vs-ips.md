# IDS vs IPS

## Overview
IDS (Intrusion Detection System) and IPS (Intrusion Prevention System) are security tools used to detect or prevent malicious activity in a network.

## IDS (Intrusion Detection System)
- Monitors network traffic
- Detects suspicious activity
- Alerts administrators
- Does NOT block traffic

## IPS (Intrusion Prevention System)
- Monitors and analyzes traffic
- Detects malicious activity
- Automatically blocks or prevents attacks
- Works inline with traffic flow

## Key Differences
| Feature | IDS | IPS |
|--------|-----|-----|
| Action | Detects | Detects + Prevents |
| Placement | Out-of-band | Inline |
| Response | Alerts | Blocks |

## What I Learned
- IDS is passive, IPS is active  
- IPS can stop attacks in real time  
- Both are essential for layered security
